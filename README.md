**Glitched Arena** is a fast-paced survival action game built as a **Telegram Mini App**. Navigate through dangerous data streams, bypass elite security walls, and take down the corrupted **CORE Boss**.

---

## 🌟 Key Features
*   ⚔️ **30+ Challenging Sectors:** Adaptive difficulty that scales with your skill.
*   👾 **Elite Boss Phases:** Encounter the MEGA_BOSS with dual turrets and orbital shields.
*   ⚡ **Cyberpunk Upgrades:** Buy better fire rate, speed, and emergency shields in the Data Shop.
*   🏆 **Global Leaderboard:** Compete with friends via Supabase-powered ranking system.
*   🎮 **Telegram Native:** Built-in Haptic Feedback and social sharing features.

## 🛠 Tech Stack
*   **Frontend:** [Phaser 3](https://phaser.io/) (Game Engine)
*   **Backend:** Python 3.11 + FastAPI + Uvicorn
*   **Database:** Supabase (PostgreSQL)
*   **API:** Telegram WebApp API (Mini Apps)
*   **Deployment:** VPS + Nginx + GitHub Actions (CI/CD)

---

## 🚀 Quick Start (Local)
1. Clone the repository:
   ```bash
   git clone https://github.com/nyutik/glitched-arena.git
   ```
2. Open `index.html` in any modern web browser.
*Note: Telegram-specific features require the game to be running inside a Telegram Bot.*
